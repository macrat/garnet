Garnet
======

Garnet is simple client for [MPD(Music Player Daemon](http://www.musicpd.org/).

install
-------
``` sh
	$ go get bitbucket.org/macrat/garnet/gr
```

usage
-----
Please see help.
``` sh
	$ gr help
```
